PUFF DADDY

A Tekken-inspired fighting game with a Kirby twist.

🎮 About the Project

PUFF DADDY is a fan-made fighting game project that combines the fast-paced, combo-driven combat of the Tekken series with the charm, adaptability, and whimsical inspiration of Kirby.

Players battle in dynamic arenas using a roster of colorful fighters capable of mastering unique fighting styles, special abilities, and character transformations. The project aims to blend competitive mechanics with accessible gameplay, creating a fun experience for both casual and dedicated fighting game fans.

Disclaimer: This project is a fan-inspired creation and is not affiliated with, endorsed by, or associated with Bandai Namco, HAL Laboratory, Nintendo, or any related companies.

✨ Features
🥊 2D arena-based fighting inspired by classic fighting game mechanics
🌟 Ability-copying and transformation systems inspired by Kirby
⚡ Combo chains, counters, and special attacks
🎭 Unique character roster with distinct fighting styles
🎮 Local multiplayer support
🏆 Training and practice modes
🎨 Stylized visuals inspired by platforming and fighting game aesthetics
🔧 Modular architecture for adding new characters and abilities
